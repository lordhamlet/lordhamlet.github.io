Random Instance Generator

Download (Windows binary): genPref2.zip

Download (source): genPref2.tar.gz

The random instance generator is constructed following Bertsimas and Demir (2002). The usage is as follows:

usage: genPref2 <number of resources> <number of tasks> <A> <C> <mode> <tightness constant> <common name>

<A> and <C> stand for the upper bound for the resource requirement and task value respectively. <mode> is either 0 or 1, indicating whether the resource requirements and capacities are correlated or not. <tightness constant> is a real number less than 1, indicating how tight the knapsack constraints are, the smaller it is, the tighter the constraint. Problem would be much harder for these tighter cases (or the solution might not even exist).
